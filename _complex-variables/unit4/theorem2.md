Suppose $p$ has no zeros on a curve $C$. Then, 

$$\int_C\frac{p'(z)}{p(z)} dz = i\arg(p(z))|_C.$$

The number of zeros is given by 

$$N = \frac{1}{2\pi i}\int_C\frac{p'}{p} dz.$$

This evaluates to 

$$N = \frac{1}{2\pi}(n\pi + 2\arg p|_\Gamma)$$
